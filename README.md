# Personal Data Science Projects

A collection of independent data science and machine learning projects covering NLP, topic modeling, anomaly detection, and automation.

**Documentation:** Each project with substantial scope includes its own `README.md`. For the news topic classification work, the full research description, methods, and usage are in [news_topic_classification/README.md](news_topic_classification/README.md).

## Projects

| Directory | Description |

| [`topic_modeling_nmf/`](topic_modeling_nmf/) | Topic modeling on CNN news articles using NMF |
| [`Wired/`](Wired/) | Exploratory analysis and scraping of Wired magazine articles |
| [`hp_tuning_gcp/`](hp_tuning_gcp/) | Hyperparameter tuning on Google Cloud AI Platform |
| [`automate_reporting/`](automate_reporting/) | Automated reporting pipeline using the News API |

| [`delete_bq/`](delete_bq/) | Cloud function for BigQuery table cleanup |
